# Lidar-Feature-Saliency-Enhancement

**Lidar Feature Saliency Enhancement** 是一个针对低成本激光雷达（如 RPLIDAR A1）在高噪声环境下几何特征识别精细度提升的研究项目。本项目通过 Python 仿真构建复杂几何场景，探索提升点云特征显著性的算法，旨在解决多机器人 SLAM 地图融合（Map Merge）中的数据关联失败问题。

## 🚀 背景与痛点

在多机器人 SLAM 过程中，当初始位置未知时，地图融合极度依赖环境的几何特征。然而，使用低成本雷达时存在以下痛点：

* **高噪声干扰**：测距误差导致几何边缘模糊。
* **低分辨率限制**：远距离物体特征点稀疏，五角星等复杂形状易退化为圆点。
* **地图融合失败**：由于特征不明显，`map_merge` 算法无法找到可靠的重叠区域，导致"乱拼图"。

本项目尝试通过算法手段增强特征点的显著性。

## 🛠️ 技术架构

项目采用模块化设计，方便扩展不同的传感器模型与增强算法：

* **`core/environment.py`**：基于 OpenCV 的高分辨率栅格地图建模，支持自定义复杂几何体（如五角星柱体）。
* **`core/sensor.py`**：模拟 RPLIDAR A1 的物理特性，包括角度分辨率、量程限制及高斯噪声模型。
* **`main.py`**：主调度程序，负责机器人路径规划（绕圈扫描）与动态可视化。
